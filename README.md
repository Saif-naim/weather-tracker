A modern, responsive Weather Tracker web application built with HTML, CSS, and JavaScript.

## ✨ Features

- 🔍 Search weather by city
- 📍 Current location weather
- 🌡️ Current temperature
- 🌡️ Max & Min temperature
- 💧 Humidity
- 🌬️ Wind Speed
- 👁️ Visibility
- ☁️ Cloud Coverage
- 📈 Air Pressure
- 🌅 Sunrise & Sunset
- 📅 Live Date & Time
- 📆 5-Day Weather Forecast
- 🌙 Dark / Light Mode
- 🎨 Dynamic Background
- ⏳ Loading Spinner
- 💾 Saves last searched city
